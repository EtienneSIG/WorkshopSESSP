# AI in the Field: Copilot, Copilot Cowork, and ClawPilot

A practical 1h00-1h15 workshop for Microsoft Solution Engineers (SE) and Azure Sales Specialists (SSP).

## Workshop Outcome
By the end of this workshop, participants will be able to:
- Explain why AI should be part of their daily workflow.
- Personalize Copilot for their role and context.
- Use Copilot Cowork patterns for team productivity.
- Apply practical daily use cases (SE and SSP tracks).
- Understand what ClawPilot is, how to access it, and when to use it (including expense reports).

## Duration and Format
- Total duration: 70 minutes (can be extended to 75 minutes with Q&A).
- Format: Live demo + hands-on checkpoints.
- Audience split: Two role-based tracks.

## Agenda (70 minutes)
0. Why use AI in your daily work (10 min)
1. How to personalize Copilot (12 min)
2. How to personalize Copilot Cowork (12 min)
3. Daily practical use cases by role (26 min)
4. What is ClawPilot: access + use cases (10 min)

See the full step-by-step plan in [docs/WORKSHOP_FACILITATOR_RUNBOOK.md](docs/WORKSHOP_FACILITATOR_RUNBOOK.md).

## Two Role-Based Paths
- SE path: technical discovery, architecture acceleration, troubleshooting, demo preparation.
- SSP path: account research, value messaging, objection handling, proposal and follow-up productivity.

Detailed activities and prompts are in [docs/PROMPT_LIBRARY.md](docs/PROMPT_LIBRARY.md).

## Suggested Preparation
- Ensure participants can access:
  - GitHub Copilot in VS Code
  - Copilot Chat
  - Copilot customization files (instructions/prompts)
  - ClawPilot access path used in your organization
- Prepare one realistic customer scenario for each role.
